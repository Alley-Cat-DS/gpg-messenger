# Code of Conduct

## Our Standards

AlleyCat GPG is a privacy and security tool. We hold the project and its
community to straightforward standards:

- Be respectful and constructive
- Focus on the technical merits of ideas
- Welcome contributors of all experience levels
- Accept that maintainers have final say on what goes in

## Unacceptable Behavior

- Harassment, personal attacks, or discriminatory language
- Sharing others' private information without consent
- Deliberate introduction of vulnerabilities or malicious code
- Spam or off-topic promotion

## Enforcement

Violations can be reported to **alleycat.elite337@passmail.net**.
The maintainer will review and respond within 72 hours.
Repeat or severe violations result in a permanent ban from the project.

## Scope

This applies to all project spaces: GitHub issues, pull requests, discussions,
and any other forum associated with AlleyCat GPG.
